# ViralStock

This is a project template for a Svelte frontend and a Golang backend.

## Prerequisites

- Docker
- Docker Compose

## Running the project

To run the project, simply run the following command from the root directory:

```bash
docker-compose up
```

This will start both the frontend and backend services.

- The Svelte frontend will be available at `http://localhost:5173`.
- The Golang backend will be available at `http://localhost:8080`.
